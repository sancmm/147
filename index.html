About CONTROL I/O¶

CONTROL I/O is an easy to use, brand independent SoftPLC, designed from the ground up for FACTORY I/O. The goal of CONTROL I/O is to provide a simple and intuitive experience for anyone starting in the world of automation. In CONTROL I/O, logic is designed with function blocks diagrams. It includes the most common functions available on real PLC so you can develop programs in the same way as you would when using professional software.
